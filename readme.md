# GenAI Backend – LangChain & LangGraph Demo

This repository demonstrates backend-ready GenAI workflows using
LLM APIs, LangChain, LangGraph, and Docker.

It is structured to showcase:
- Safe LLM API integration
- Stateful conversational logic
- Deterministic DAG-based orchestration
- Conditional routing between tools and LLMs
- Deployment readiness using Docker

The code is intentionally simple, explicit, and production-oriented.
